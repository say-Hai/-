### 初始化数组

```c++
//当需要初始化数组时
	int a[10] = {0}
//代表将数组a的元素全部初始化0
//但是
	int a[10] = {1}
//代表 a[0] = 1,其余元素初始化为0 
//类似地
	int a[10] = {1, 5, 2} //前三个元素分别赋值，后面元素赋为0
```

### 输入流读入数据

#### 	**读入一个数据时**

```c++
//当读入的数据有多行时，记得输入换行符，提示程序第一行输入完毕，不然在读取下一行时会出错。
	scanf("%d \n", digit);
//当使用cin时，我们读取下一行之前需要清除缓存流。
	cin>>digit;
	cin.ignore();
//不然的话，后面使用其他读入一行的函数会有各种各样的问题
/*
	注意：这两个函数读到空格或者换行符就会停止，要读入包含空格的数据则需要其他函数
*/
----------------------------------------------------------------------------------------------------------------
//一次从输入流读入一个字符时，使用范围：适合一个个处理字符的场景
	char s = getchar();//与之配对的->putchar(s)：输出一个字符
/*
	注意：当从控制台输入时，需要按一下ctrl + z，再按一次回车才能结束输入。
	在程序中通过 s == EOF 来判断输入结束。
*/

```
#### 	**读入一行数据时**

> 当一次读入一行，且需要包含空格时

```c++
//方法①：通过字符数组来进行读取
	fgets(s, sizeof(s), stdin);
//参数1：字符数组 char类型；
//参数2：数组长度，即指定最大读入数量
//参数3：输入流 stdin
----------------------------------------------------------------------------------------------------------------
//方法②：通过字符串来读取一行
    getline(cin,str);
//参数1：输入流
//参数②：字符串string

```

### 从字符串读入数据

```c++
//当我们将输入流的数据读入到char数组中时，想要从字符串中读出我们想要的数据
	sscanf(s, "%d %d", &a, &b);//从字符数组读入两个int型变量
//参数1：从s读出我们要的数据
----------------------------------------------------------------------------------------------------------------
//同理，当我们要将数据输入到字符串时
    sprintf(ans[i], "%d+%d=%d", a, b, c);
//参数1：将要写入数据的字符串
//参数2：将要写入的数据
//参数3：参数2中占位符所代表的数据
```
